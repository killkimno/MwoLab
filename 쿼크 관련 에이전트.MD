# 쿼크 관련 에이전트 규칙

## 정보 탭과 통계 탭 쿼크 표시

- 정보 탭 쿼크 카드의 쿼크 서머리 하위 항목과 통계 탭의 `QUIRKS` 하위 버튼은 표시 대상이 거의 같다.
- 정보 탭의 쿼크 하위 서머리들은 통계 탭의 `QUIRKS` 하위 버튼들과 이름과 기준을 최대한 맞춘다.
- 단, 일부 규칙은 다르다.
- 정보 탭에는 듀레이션을 별도 서머리로 표시한다.
- 쿨 다운은 정보 탭과 통계 탭의 구성이 서로 다를 수 있다.
- 정보 탭의 사거리 서머리에는 센서 값을 포함한다.
- 통계 탭에 있는 쿼크 하위 버튼 이름은 정보 탭 `쿼크 서머리` 하위 목록 이름들과 맞춘다.
- 통계 탭의 `전체` 버튼 이름은 변경하지 않는다.
