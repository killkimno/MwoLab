# MwoLab

MechWarrior Online 메크 스펙 비교 사이트입니다.

## 주소

https://killkimno.github.io/MwoLab/

## 소개

MwoLab은 MechWarrior Online의 메크 스펙을 빠르게 확인하고 비교하기 위한 웹 사이트입니다.

- 메크별 내구도, 기동성, 쿼크 정보를 확인할 수 있습니다.
- 여러 메크의 스펙을 한 화면에서 비교할 수 있습니다.
