# 무기 데미지 규칙

- 무기의 직격 데미지는 `stats.damage * stats.numFiring`으로 계산한다.
- `stats.splashPercent`가 0보다 크면 직격 지점에 인접한 두 부위에 각각 `직격 데미지 * splashPercent`의 스플래쉬 데미지가 적용된다.
- 스플래쉬를 포함한 총데미지는 `직격 데미지 + 스플래쉬 데미지 * 2`로 계산한다.
- 툴팁의 DAMAGE는 `직격 + 스플래쉬 합계` 형식으로 표시하고 스플래쉬 합계 부분만 노란색으로 표시한다. 예: C-ER PPC는 `10 + 5`로 표시한다.
- 데미지 표시는 정수일 때 소수점을 붙이지 않고, 소수가 발생했을 때만 소수 첫째 자리까지 표시한다.
- 머신건, 빔레이저, 화염방사기, 로터리 AC는 ROF 계열로 취급하고 툴팁 DAMAGE를 초당 데미지(`/s`)로 표시한다.
- 머신건과 로터리 AC의 초당 데미지는 `직격 데미지 * stats.rof`로 계산하며 ROF 쿼크도 반영한다. 예: 로터리 AC/5는 `1.1 * 10 = 11/s`로 표시한다.
- 빔레이저와 화염방사기의 연속형 `stats.damage`는 초당 데미지 값이므로 별도로 ROF를 곱하지 않고 `/s`를 붙인다.
- 무기 툴팁의 사거리 그룹은 `MIN RANGE`(최소 사거리가 있을 때만), `OPTIMAL RANGE`, `MAX RANGE`, `VELOCITY` 순서로 표시한다.
- `MIN RANGE`는 첫 구간의 데미지 배율이 최대 배율보다 낮고 이후에 최대 데미지가 시작되는 무기에만 표시한다. 예: LRM은 180m를 표시한다.
- 사거리 증가 쿼크는 `MIN RANGE`에 적용하지 않는다. 시뮬레이션에서도 최소 사거리 경계는 그대로 두고 그 이후의 적정·최대 사거리 경계만 증가시킨다.
- ATM은 60~349m에서 1.25배, 350~649m에서 1배, 650~1100m에서 0.8배의 계단식 데미지를 적용한다. 사거리 쿼크가 있으면 350m 이후의 구간 경계와 최대 사거리만 증가하고 최소 사거리 60m는 유지한다.
- ATM은 특수 스탯 그룹에 `DAMAGE BAND 1~3`을 표시하며 각 행은 `최종데미지 (시작~끝 m)` 형식을 사용한다.
- `stats.spread`가 0보다 크면 사거리 그룹 다음의 별도 그룹에 `SPREAD`를 표시하며, 일반·무기 계열·무기별 퍼짐 감소 쿼크를 반영한다.
- `stats.critChanceIncrease`에 0이 아닌 값이 있으면 같은 후속 그룹에 `CRITICAL CHANCE`를 퍼센트로 표시한다. 여러 단계 값은 `14% / 8% / 3%` 형식으로 모두 표시하고, `-1`은 절대 발생하지 않는 단계이므로 빨간색 `X`로 표시한다. 예: 가우스 라이플은 `17% / X / X`로 표시한다.
- `stats.critDamMult`가 존재하고 값이 1이 아니면 `CRITICAL CHANCE` 바로 다음 행에 `CRITICAL DAMAGE`를 배율로 표시한다. 1 미만의 감소 배율은 빨간색으로 표시하고 값이 1이면 행을 표시하지 않는다.
- 화염방사기와 클랜 플라즈마 PPC는 특수 스탯 그룹에 적에게 가하는 발열을 `TARGET HEAT`로 표시한다. 연속형 화염방사기는 `/s` 단위를 사용한다.
- `stats.chargeTime`이 0보다 큰 가우스 라이플 계열은 특수 스탯 그룹에 `CHARGE TIME`을 초 단위로 표시한다.
- 울트라 AC는 `JAM DURATION`, `JAM CHANCE`, `EXPECTED DPM`을 특수 스탯 그룹에 표시하고 관련 쿼크를 반영한다.
- 울트라 AC 기대 분당 데미지는 쿨다운마다 가능한 추가 발사를 모두 시도한다고 가정한다. 더블탭 잼 시 추가탄은 발사되지 않으며, 기존 쿨다운은 초기화하지 않고 잼 지속시간 동안 다음 발사를 막는다.
- 울트라 AC 한 주기의 기대 피해는 `기본 피해 * (1 + (1 - 최종 잼 확률) * ShotsDuringCooldown)`이다. 기대 주기는 비잼 시 최종 쿨다운, 잼 시 `max(최종 쿨다운, volleydelay + 최종 잼 지속시간)`을 잼 확률로 가중하며, `기대 피해 / 기대 주기 * 60`으로 `EXPECTED DPM`을 계산한다.
- 빌드의 알파 데미지는 스플래쉬를 포함한 총데미지를 사용한다.
- DPS 시뮬레이션은 스플래쉬 적용 토글을 제공하며 기본값은 활성화한다. 비활성화하면 직격 데미지만 계산한다.

## 멕랩 장비 목록 필터

- 선택한 멕의 실효 하드포인트에 없는 종류의 무기는 멕랩 장비 목록에 표시하지 않는다.
- 일반 멕은 기본 스펙의 하드포인트를 사용하고, 옴니멕은 현재 장착한 옴니포드의 하드포인트만 사용한다. 하드포인트가 없는 옴니포드에 기본 포드의 하드포인트를 대신 적용하지 않는다.
- 에너지, 미사일, 발리스틱, AMS 무기는 각각 대응하는 실효 하드포인트가 하나 이상 있을 때만 표시한다.
- ECM은 일반 멕의 `CanEquipECM` 스펙 또는 ECM 하드포인트가 있을 때 표시한다. 옴니멕은 현재 옴니포드 구성에 ECM 하드포인트가 있을 때만 표시한다.
- 점프젯은 `MaxJumpJets`가 0보다 크고 해당 점프젯의 중량 범위와 멕 중량이 맞을 때만 표시한다.
- MASC는 `CanEquipMASC` 또는 `CanEquipMasc`가 활성화되어 있고 해당 MASC의 중량 범위와 멕 중량이 맞을 때만 표시한다.
